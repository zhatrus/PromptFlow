# Project Context

## Project Name
**[TBD]** — Посібник з форматування промптів  
_(варіанти: PromptUA, PromptGuide, Промптер — без літери "Z"!)_

## Purpose
PWA-застосунок — інтерактивний довідник з форматування промптів для роботи з ШІ.
- Повний курс (7 частин) з технік форматування
- 15+ готових промптів-хелперів (копіюй → використовуй)
- Бібліотека промптів спільноти (з голосуванням)
- Квізи для перевірки знань
- Особисті збережені промпти

**Фаза 1:** Автономний PWA (офлайн-режим, локальне збереження)
**Фаза 2:** SaaS з бекендом (авторизація, хмарна синхронізація, ШІ-функції)

## Users
- Початкові: розробники, маркетологи, контент-мейкери (~500 користувачів)
- Цільові: будь-хто, хто працює з ChatGPT/Claude/інші LLM
- Потенційно: B2B для корпоративного навчання

## User Roles (Фаза 2)
- **Guest** — читання курсу, використання хелперів
- **User** — збереження промптів, голосування, створення приватних промптів
- **Contributor** — створення публічних промптів у бібліотеку
- **Admin** — модерація, статистика

## Non-goals (Фаза 1)
- НЕ чат-бот / НЕ інтерфейс для LLM API
- НЕ генератор промптів (тільки навчання + шаблони)
- НЕ платний контент / paywall
- НЕ автентифікація (тільки локальне збереження)

## Tech Stack
- **Frontend:** Next.js 14+ (App Router), React 18
- **Styling:** TailwindCSS + shadcn/ui
- **Icons:** Lucide React
- **PWA:** next-pwa, Service Worker, Web App Manifest
- **Local Storage:** IndexedDB (Dexie.js)
- **Search:** Fuse.js (client-side fuzzy search)
- **State:** Zustand (lightweight)

### Фаза 2 (майбутнє)
- **Auth:** NextAuth.js або Supabase Auth
- **Database:** PostgreSQL (Supabase)
- **AI Integration:** OpenAI API / Claude API (опціонально)
- **Deployment:** Vercel або Docker (Portainer)
